## 3. GTM 전략

### Phase 0: 프로덕트 완성 (Month 4~5)

```
목표: 10개 Founding Member와 2개월 제품 개선
조건: 1년 완전 무료

모집: 인스타 DM + 지인 + 지역 방문
선정: 인스타 DM 주문, 일 5건+ 문의, 피드백 적극적, 메뉴 구조 다양

운영:
- 온보딩 직접 지원 (화면 공유/방문)
- 카톡 단톡방, 주 1회 피드백
- 매주 업데이트 배포

혜택:
✅ 1년 무료 + 신기능 우선 + 기능 요청 우선
✅ 1년 후 영구 50% 할인
→ 조건: 월 1회 피드백, 사례 활용 동의

마일스톤:
M1: 주문 플로우 안정
M2: AI 90%+ 정확, 8/10 "없으면 못 살아요"

비용: 월 75,000원 × 2개월 = 약 15만원
```

### Phase 1: 마케팅 시작 (Month 6~)

```
전제: 제품 완성도 확인, 10개 가게 실제 데이터

채널:
1. 고객 성공 사례 콘텐츠 (가장 중요)
   "해피케이크 사장님, 하루 2시간 절약, 야간 주문 30% 증가"
2. 인스타 공식 계정
   사례40 / 공감30 / 시연20 / 팁10 / 주 3~4회
3. 네이버 블로그 SEO
   "케이크 주문 관리", "인스타 주문 자동화"
4. 추천 프로그램
   1명 추천 → 다음 달 무료 / "오대리 쓰는 가게" 뱃지
```

| 캔들 | option_sum | type, scent, size, container | 선택 | — |
| 디저트 | option_sum | flavor, quantity, box | 불필요 | — |
| 떡케이크 | option_sum | size, flavor, deco, lettering | 불필요 | 케이크 재활용 |
| 애견미용 | option_sum | breed, size, cut, bath | 필수 | 건강 수집 |
| 네일아트 | area_based | hand, art_type, length | 선택 | 손 10개 |
| 네온사인 | option_sum+char | text, font, color, size | 선택 | 글자수×단가 |
| 사진관 | option_sum | shoot, person, print | 불필요 | — |
| 자수/각인 | option_sum+char | material, text, font | 불필요 | 글자수×단가 |
| 반려동물초상화 | option_sum | pet, style, size, frame | 선택 | 수량별 할인 |

### 견적 공식

```
option_sum:        base + Σ(option)
option_sum×qty:    (base + Σ(option)) × qty
option_sum+char:   base_per_char × chars + Σ(option)
tier_based:        tier.price + Σ(option)
area_based:        base_per_unit × units + Σ(option)
```
