## 1. Google Sheets 웨이팅리스트 연동

### 배경
- 베타 신청 폼 데이터가 `console.log`만 찍히고 저장되지 않던 상태
- Google Apps Script 웹앱을 경유하여 Google Sheets에 자동 저장되도록 연동

### 흐름
```
[브라우저 폼 제출] → /api/beta-signup (서버) → Apps Script URL (서버→서버) → Google Sheets
```

### 변경 파일
| 파일 | 변경 내용 |
|------|-----------|
| `src/app/api/beta-signup/route.ts` | `console.log` → Apps Script URL로 fetch POST |
| `.env.local` (신규) | `GOOGLE_SHEET_WEBHOOK_URL` 환경변수 |

### 동작 방식
- 환경변수 설정 시: Apps Script로 POST → Google Sheets에 행 추가
- 미설정 시: 기존처럼 콘솔 로그만 기록 (개발 환경 호환)
- Sheets 저장 실패해도 사용자에게는 성공 응답 (UX 우선, 서버 로그만 `console.error`)

### Google Sheets 정보
- 스프레드시트: `1DT21bbaKreGh46h5o9eXvUKtvD7q26HeVVpJWHK9G50`
- 헤더: 타임스탬프 | 성함 | 가게명 | 연락처 | 이메일 | 업종 | 기능요청
